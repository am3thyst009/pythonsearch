# 🎌 Консольный поиск фильмов, аниме и игр

Консольное приложение для поиска по трём источникам одновременно:
- 🎌 **Аниме** — Jikan API (MyAnimeList), без ключа
- 🎬 **Фильмы** — OMDB API
- 🎮 **Игры** — RAWG API

## Технологии

| Тема | Где используется |
|---|---|
| **Асинхронность** (`asyncio`, `aiohttp`) | Параллельные запросы к трём API в `search_all()` |
| **Декораторы** | `@cache`, `@timer`, `@retry` — `decorators.py` |
| **БД SQLite** | История и избранное — `db.py` |
| **Парсинг JSON** | `parse_anime / parse_movies / parse_games` — `api.py` |
| **Логирование** | `logging` в файл `app.log` + консоль — `logger.py` |
| **Конфигурация** | API-ключи в `.env` через `python-dotenv` |
| **Экспорт CSV** | Избранное → `favorites.csv` через `csv.DictWriter` |

## Установка и запуск

```bash
pip install -r requirements.txt
cp .env.example .env   # вставьте свои ключи в .env
python main.py
```

## API-ключи (бесплатно)

- **OMDB**: [omdbapi.com/apikey.aspx](https://omdbapi.com/apikey.aspx)
- **RAWG**: [rawg.io/apidocs](https://rawg.io/apidocs)
- **Jikan** (аниме): ключ не нужен

## Структура проекта

```
├── main.py         # Точка входа, консольный интерфейс, главный цикл
├── api.py          # Async-запросы к API, парсинг JSON, фильтры, пагинация
├── db.py           # SQLite: история, избранное, статистика, экспорт CSV
├── decorators.py   # Декораторы: @cache, @timer, @retry
├── logger.py       # Настройка logging (файл + консоль)
├── .env.example    # Шаблон конфига с ключами
├── requirements.txt
└── data.db         # Создаётся автоматически при первом запуске
```

## Возможности

- Поиск сразу по трём источникам параллельно
- Фильтрация по жанру и году выпуска
- Пагинация результатов (кнопка "следующая страница")
- Добавление в избранное, удаление
- История всех поисковых запросов
- Статистика: топ запросов, разбивка по источникам
- Экспорт избранного в CSV
- Кэширование повторных запросов (5 минут)
- Автоповтор при ошибке сети (3 попытки)
- Логирование всех событий в `app.log`
